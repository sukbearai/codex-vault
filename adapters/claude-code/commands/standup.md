Run the morning standup:

1. Read `Home.md` for current state
2. Read `brain/North Star.md` for goals
3. Check `work/Index.md` for active projects
4. Check recent git activity: `git log --oneline --since="24 hours ago" --no-merges`

Present a structured summary:
- **Yesterday**: What got done (from git log)
- **Active Work**: Current projects in work/active/
- **North Star Alignment**: How active work maps to goals
- **Suggested Focus**: What to prioritize today

Keep it concise — quick orientation, not a deep dive.
